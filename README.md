# Dashboard Logístico - Power BI

## Descrição
Dashboard para monitoramento e análise de processos logísticos.

## Indicadores
- Volume de movimentações
- Entradas e saídas
- Eficiência operacional

## Funcionalidades
- Controle logístico
- Identificação de gargalos
- Otimização de processos

## Ferramenta
- Power BI

## Objetivo
Melhorar eficiência e controle operacional.
